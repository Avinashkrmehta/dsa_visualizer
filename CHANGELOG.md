# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-18

### Added
- Initial release of DSA Visualizer
- Complete curriculum with 12 sections covering 50+ topics
- Interactive CLI with menu-based navigation
- Progress tracking system
- Comprehensive learning modules:
  - Fundamentals (Complexity, Memory, Pointers, Recursion)
  - Basic Data Structures (Array, String, Linked List)
  - Stack & Queue implementations
  - Hashing concepts
  - Trees (Binary Tree, BST)
  - Heaps
  - Graphs
  - Sorting algorithms
  - Searching algorithms
  - Advanced algorithms (DP, Greedy, Backtracking)
  - String algorithms
  - Advanced data structures
- Ruby vs C++ code comparisons for every topic
- Visual ASCII representations
- Memory layout visualizations
- Time and space complexity analysis
- Detailed notes and key points for each topic
- Common mistakes and best practices
- Real-world applications
- Practice problems
- NotesManager for formatted output
- MemoryTracker for operation tracking
- Comparator for language comparisons
- Visualizer for ASCII art and formatting

### Documentation
- Comprehensive README.md
- Detailed USAGE.md guide
- QUICKSTART.md for beginners
- LICENSE.txt (MIT)
- Example scripts in examples/

### Features
- Command-based topic selection (e.g., "2.1" for Arrays)
- Progress tracking saved to user home directory
- Color-coded output for better readability
- Step-by-step operation walkthroughs
- Side-by-side Ruby and C++ comparisons
